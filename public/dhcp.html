Instalación y configuración del Servidor DHCP

💥 Paso 1

Luego de tener instalado el SO Windows Server 2019 en la máquina virtual se verifica que la configuración de red se encuentre en los valores Red interna

💥 Paso 2

Accedemos a la configuración de red del sistema operativo y asignamos la dirección IP de forma estática. Para nuestro caso 192.168.100.2 para posteriormente tener un rango de direcciones completo. La máscara de red se asigna de la misma manera y establecemos la puerta de enlace predeterminada. También configuramos las direcciones de servidor DNS, las cuales pueden ser configuradas a voluntad del usuario dependiendo los DNS que desee utilizar.

💥 Paso 3

Ahora verificamos que los cambios hayan sido aplicados correctamente con el comando ipconfig desde la terminal de comandos del sistema operativo.

💥 Paso 4

Nos ubicamos en el administrador del servidor y procederemos a configurar este servidor local añadiendo un nuevo rol como se muestra a continuación.

💥 Paso 5

Se instala o añade un nuevo rol para lo que seleccionamos la opción Instalación basada en características o roles y hacemos clic en Siguiente.

💥 Paso 6

Para nuestro caso, seleccionamos el servidor con nombre WIN-Q7LNPA2IIGO que es donde queremos añadir nuestro nuevo rol el cual nos permitirá asignar direcciones IP de manera dinámica.

💥 Paso 7

Ahora procedemos a buscar el rol Servidor DHCP y seleccionarlo y hacemos clic en Siguiente

Desde la siguiente ventana, confirmamos la instalación del rol.

💥 Paso 8

Ahora se configura el rol recientemente instalado dando clic en la banderita o directamente en la opción llamada DHCP

💥 Paso 9

En esta ventana se aprecia toda la configuración que brinda el rol DHCP y vamos a comenzar por agregar un nuevo ámbito dando clic derecho sobre la opción IPv4

💥 Paso 10

Ahora asignamos un nuevo nombre a decisión propia y una descripción que permita identificar de manera rápida el ámbito que estamos creando.

💥 Paso 11

Luego configuramos un intervalo de direcciones IP que va a ser capaz de asignar nuestro DHCP a los distintos dispositivos que se conecten a este servidor. Además se establece la máscara de subred y su respectiva longitud. Hacemos clic en Siguiente

💥 Paso 12

A continuación si se desea pueden agregarse intervalos de direcciones IP que estén excluidas del servidor. O en su defecto, si deseamos excluir una sola dirección en concreto. Para nuestro caso simplemente daremos clic en Siguiente sin excluir ningún tipo de dirección IP

💥 Paso 13

En el siguiente paso se configura la duración de la concesión, es decir, cuánto tiempo puede un cliente usar la dirección IP en este ámbito. Para nuestro caso lo dejaremos por defecto y haremos clic en Siguiente

💥 Paso 14

Luego agregamos la puerta de enlace que había sido configurada anteriormente con la dirección 192.168.100.1. Damos clic en Agregar y luego dar clic en Siguiente hasta aceptar la opción Activar este ámbito ahora sin ninguna configuración adicional.

💥 Paso 15

De esta manera, procedemos a dar clic derecho en la opción IPv4 y reiniciamos el servidor. Luego de esto, nuestro servidor DHCP quedará instalado y configurado para ser utilizado.

💥 Visualización y conexión desde Windows Client al servidor DHCP instalado y configurado.

Primero revisamos la configuración de red de la máquina cliente con sistema operativo Windows 10 y verificamos que se encuentre en el mismo segmento de red de nuestra máquina servidor donde se está ejecutando el servidor DHCP.
Si esto coincide, procedemos a verificar que se esté recibiendo la dirección ip de manera dinámica y confirmamos esta configuración.

Luego, procedemos a acceder a la terminal de comandos del sistema operativo y verificamos que nuestra máquina haya recibido la dirección IP de acuerdo a la configuración de nuestro servidor DHCP.

En caso de que no se haya asignado la dirección IP correctamente podemos reiniciar la máquina para que efectúe las configuraciones de manera correcta.
